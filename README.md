# power-bi-portfolio-project
Business Intelligence Dashboard Suite - Employee Expense &amp; iPhone Sales Analysis
## 📌 Overview
A comprehensive Business Intelligence portfolio featuring two end-to-end Power BI dashboards across Finance and Sales domains.

## 📂 Project Modules

### 1️⃣ Employee Expense Reimbursement Dashboard (Finance)
- **Objective:** Track and analyze employee expense claims across projects
- **Key Features:**
  - Data cleaning & standardization using Power Query
  - Currency conversion (USD/EURO → INR) with conditional logic
  - DAX measures for real-time reimbursement tracking
  - Interactive slicers for project/employee filtering
- **Impact:** 90% reduction in manual reporting time

### 2️⃣ iPhone 14 vs 15 Sales Comparison Dashboard (Sales)
- **Objective:** Compare global sales performance and identify market trends
- **Key Features:**
  - Multi-source data integration (CSV files)
  - YoY variance analysis (absolute & percentage)
  - Dynamic TOP 10 country filtering
  - Conditional formatting for instant insights
- **Impact:** Identified high-growth markets and underperforming regions

## 🛠️ Tools & Technologies
- **Power BI Desktop** - Dashboard development
- **Power Query (M Language)** - Data transformation & cleaning
- **DAX** - Custom measures & calculations
- **Data Modeling** - Star schema design
- **Excel/CSV** - Data sources
